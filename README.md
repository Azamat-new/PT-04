1) для установки джанго 

pip install django
pip install dotenv

2) Вдруг если нет виртуального окружения
python -m venv venv или python3 -m venv venv
для MacOS - source venv/bin/activate
для Windows - .venv/Scripts/activate

3) Команды для создание проекта на джанго

- django-admin startproject codify - для создание джанго-проекта 
- ls - для проверки директории 
- cd codify - cd команда для открытия -> codify название проекта
- python manage.py runserver - запуск сервера
- сtrl + c - остановка сервера
- python manage.py startapp lms - для создания приложения
- pip freeze > requirements.txt  - для фиксации зависимостей
